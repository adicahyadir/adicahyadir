# 🚀 Frontend Developer & UI/UX Designer Roadmap

*A Practical & Professional Learning Path (2026 Edition)*

---

## 🧭 Overview

Roadmap ini dirancang untuk membantu kamu menjadi **Frontend Developer + UI/UX Designer** yang siap kerja dengan pendekatan bertahap, fokus, dan berbasis project.

> ⚠️ Prinsip utama: *Depth over breadth* — kuasai satu hal, baru lanjut ke berikutnya.

---

# 🧱 1. Foundation (4–6 Minggu)

## 🎨 UI/UX Basics

Pelajari prinsip dasar desain:

* Visual hierarchy
* Typography
* Color theory
* Spacing & layout
* UX flow & wireframing

### Tools:

* Figma → https://www.figma.com/
* FigJam → https://www.figma.com/figjam/

### Resources:

* https://www.refactoringui.com/
* https://lawsofux.com/
* https://uxdesign.cc/

### Output:

* Wireframe & mockup landing page di Figma

---

## 💻 HTML & CSS Basic

Pelajari:

* Semantic HTML
* CSS fundamentals
* Flexbox & Grid

### Resources:

* https://developer.mozilla.org/en-US/docs/Web
* https://css-tricks.com/
* https://web.dev/learn/css/

### Practice:

* Build static landing page dari design Figma

# ⚠️ Common Mistakes

❌ Belajar semua framework sekaligus
❌ Fokus tools, bukan konsep
❌ Tidak membuat project

✅ Fokus 1 stack
✅ Build real projects
✅ Konsisten belajar tiap hari
